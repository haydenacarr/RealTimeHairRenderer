# Real-Time Hair Renderer
This is a Hair Renderer written in DirectX 12

---

## Features

- **Kajiya-Kay Lighting
- **Marschner Shading
- **MSAA

---

### API's and Libraries
- `DirectX 12` for Graphics API
- `SDL 2` for Windowing
- `tinyobjloader` for Loading OBJ Files

---

## Requirements

- Windows 10/11
- DirectX 12.1 Capable Graphics Card

---

## Controls

- `w` — Move Forward  
- `a` — Move Left
- `s` — Move Backward 
- `d` — Move Right
- `↑` (Up Arrow) — Look Up
- `←` (Left Arrow) — Look Left
- `↓` (Down Arrow) — Look Down
- `→` (Right Arrow) — Look Right
